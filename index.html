发表于2023-09-04
冒泡排序-原理两套循环： n个数从头开始依次比较，如果前一个数比后一个数大，那么交换次序==重点==：比较的是位置上的数字，每次循环都相对位置而言 n个位置数依次重复上面的动作，除了最后一个 每比较一轮，重复的比较次数就少一 代码实现： function f1(arr){ for(var i=0;i<arr.length-1;i++){//上面的2,3 for(var j=0;j<arr.length-1-i;j++){//上面的1 if(arr[j]>arr[j+1]){ var t; t=arr[j]; arr[j]=arr[j+1]; arr[j+1]=t; } } } return arr; }
