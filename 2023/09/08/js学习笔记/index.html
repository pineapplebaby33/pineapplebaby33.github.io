1. 变量提升不进行赋值操作  
2. 函数提升不进行调用操作

##-对象
####什么是对象  
对象是复杂数据类型Object类  
本质：一组无序的相关属性和方法的集合
####为什么要创建对象  

结构表达更明确

对象包含：==属性== ==方法==


####创建对象的三种方式  
#####1. 对象字面量  {}
```javascript
var obj = {}//基本的空对象
var obj = {
    name: '李四',
    age: 18,
    f1: function(){
        console.log('hi~');
    }
}

注意：
1. Object() O大写 加小括号

4. 匿名函数

####new的执行过程  
1. 在内存里创建一个新的空对象
2. 让this指向这个对象
3. 执行构造函数里的代码，让新对象添加属性和方法
4. 返回这个对象（构造函数没有return 的原因）
